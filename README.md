**Installation guide:**
1. Extract [``MelonLoader.x64.zip``](https://github.com/LavaGang/MelonLoader/releases/download/v0.7.2/MelonLoader.x64.zip) contents into ``root folder`` of the game (where armaphract.exe is located)
2. Launch game once, letting it load into main menu, then close the game. It will create mods folder in root.
3. Extract [``BetterHUD.zip``](https://github.com/Sannanore/Armaphract-BetterHUDMod/releases/download/v0.1.0/BetterHUD.zip) into ``Mods`` folder in ``root folder``

⚠️ Don't forget to check release notes for tested game version compatability.
